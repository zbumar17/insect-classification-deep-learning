# Deep Learning-Based Insect Genera Classification

This repository contains the implementation of my MSc graduation thesis
on insect image classification using deep learning techniques.

## Insect Genera Classified
The model classifies images into the following seven insect genera:
- Aedes
- Culex
- Anopheles
- Tritoma
- Pediculus
- Cimex
- Ctenocephalides

## Models Implemented
- Custom Convolutional Neural Network (CNN) from scratch
- Transfer Learning models:
  - ResNet50
  - MobileNet
  - SqueezeNet
  - EfficientNet
- Hybrid Ensemble Model combining the best-performing transfer learning models

## Framework and Tools
- Python
- PyTorch
- Google Colab
